# STM32-FreeRTOS-Blink-Button
Цей проєкт демонструє роботу операційної системи реального часу FreeRTOS на мікроконтролері STM32 (платформа Black Pill / F411). Програма реалізує паралельне виконання двох задач: керування світлодіодами та динамічне опитування кнопки.

### Функціонал:
**LED_Task**: Почергово перемикає 3 світлодіоди на портах PA1, PA2, PA3.

**ButtonTask**: Опитує кнопку на PA0 (System Button). При натисканні змінює швидкість мигання (4 режими).

### Як запустити:
1. Відкрийте файл `.ioc` у STM32CubeIDE.
2. Згенеруйте код (якщо потрібно).
3. Скомпілюйте та прошийте через ST-Link.
